

### 使用说明

该项目使用 vscode 配合 platformio 插件作为开发环境，使用前需要在 vscode 的扩展商店中安装 platformio 插件，然后右键点击项目文件夹，选择 “通过 Code 打开” ，或者在 vscode 中点击 “文件” -> "打开文件夹"，选择项目文件夹。
